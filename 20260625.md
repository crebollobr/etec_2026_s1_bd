# Avaliação 25/06/2026 — Criação de Funções e Procedures no SQL

**Duração:** 2 horas

## Conteúdo

Dois objetos criados pelo usuário, contrastando **função** (retorna **um valor escalar**) e **procedure** (pode retornar **uma linha/conjunto de resultados**).

> ⚠️ No MySQL, `CREATE FUNCTION` só retorna um valor escalar (`INT`, `VARCHAR`, `DATE`...). Para retornar uma linha/tupla ou um resultado, usa-se `CREATE PROCEDURE` (com `SELECT` interno ou parâmetros `OUT`).

### 1. Função com Retorno de Valor (`CREATE FUNCTION`) — ~1h
- Sintaxe básica `CREATE FUNCTION ... RETURNS <tipo>`
- Parâmetros de entrada e `RETURN`
- Uso de `DELIMITER`
- Exemplo: função simples de manipulação de texto (ou cálculo simples)

### 2. Procedure com Retorno de Linha/Tupla (`CREATE PROCEDURE`) — ~1h
- Diferença entre **função** (1 valor) × **procedure** (linha/resultado)
- Sintaxe `CREATE PROCEDURE` e chamada com `CALL`
- Retornar uma linha: via `SELECT` interno **ou** parâmetros `OUT`
- Exemplo prático
- Exercícios
